# `@extension-jobs/cli`

Local administration CLI for an Extension Jobs checkout. Node.js 24+ is required.

```sh
npm install -g @extension-jobs/cli
cd extension-jobs
extension-jobs init
extension-jobs doctor
extension-jobs start
```

Set `EXTENSION_JOBS_HOME` when invoking the CLI outside the checkout. Resume imports are copied into a private local vault; secret values generated by `init` are written to `.env` and never printed.
