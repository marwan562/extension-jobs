# Profile and fact model

A canonical candidate profile contains facts with stable IDs, values, provenance, confidence, verification status, and timestamps. Imported text is evidence, not permission. Facts must be reviewed and verified before deterministic filling or tailoring.

Tailored resumes and applications reference immutable profile snapshots and the exact fact IDs used. Legal, demographic, disability, clearance, sponsorship, work authorization, relocation, salary, and similarly sensitive answers are never inferred from ordinary resume text.
