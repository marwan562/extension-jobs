# OpenClaw skill

The publishable `@extension-jobs/openclaw-jobs` package bundles `skills/extension-jobs/SKILL.md`. The skill teaches capability-first planning, verified-fact grounding, privacy, challenge/manual-handoff behavior, campaign safety, emergency stop, and the rule that agents may request but never grant approval.

Generic focused tools are preferred. Deprecated `wuzzuf_*` aliases are kept for migration. Tool schemas default preparation to dry-run, omit raw token/resume/cookie fields, and describe unsupported capabilities truthfully.
